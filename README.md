# dotfiles

preferences of zsh, vim, ...

## applications

* zsh
    * zshrc -> ~/.zshrc
    * zcomp/* -> ~/.zcomp
* vim
    * vim/* -> ~/.vim
* latexmk
    * latexmkrc -> ~/.latexmkrc

## installation

* `> ./install.sh`

## notice

Some preferences are supposed to be used in combination with [frontemare](https://github.com/tetutaro/frontemare).
